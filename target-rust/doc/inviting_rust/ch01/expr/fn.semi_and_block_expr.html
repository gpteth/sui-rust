§分号表达式 vs 块表达式

  1. 分号表达式返回值永远为自身的单元(Unit)类型：()
  2. 分号表达式只有在块表达式最后一行才会进行求值，其他时候只作为「连接符」存在
  3. 块表达式只对其最后一行表达式进行求值。
fn main(){
    ;
    ;
    {
        ()
    }
    {
        ();
        use std::vec::Vec;
    }
    ();
    &();
    &{;}; // -> &()
    ; // ->  ()
}
